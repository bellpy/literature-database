### Getting started

1. Clone the repo
   ```sh
   git clone https://github.com/bellpy/literature-database.git
   ```
2. Install python library dependecies
   ```sh
   pip install -r requirements.txt
   ```
3. Run the application
   ```sh
   python3 main.py
   ```
